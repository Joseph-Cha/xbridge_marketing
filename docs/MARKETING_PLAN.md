# XBridge 홍보 페이지 기획안 (사전 수요 조사용)

> 작성일: 2026-03-05
> 버전: 2.0
> 목적: 서비스 출시 전 잠재 고객 수요 조사 및 리드 수집

---

## 1. 프로젝트 개요

### 1.1 목적
서비스 정식 출시 전, 잠재 고객의 **이용 의향을 파악**하고 **기업 담당자 정보를 수집**하여 추후 마케팅 및 서비스 제안에 활용한다.

### 1.2 페이지 목표
| 목표 | 설명 |
|------|------|
| **1차 목표** | 잠재 고객 연락처 정보 수집 (리드 확보) |
| **2차 목표** | 서비스에 대한 관심도 및 시장 수요 검증 |
| **3차 목표** | 타겟 고객의 니즈 및 선호 기능 파악 |

### 1.3 타겟 고객
| 구분 | 내용 |
|------|------|
| **업종** | 식품 제조업 (가공식품, 음료, 건강식품, 전통식품 등) |
| **규모** | 중소기업 (연매출 10억~500억 규모) |
| **특징** | 해외 판로 개척 의지가 있으나 수출 경험/인프라 부족 |
| **니즈** | 복잡한 수출 절차 간소화, 신뢰할 수 있는 바이어 연결 |

### 1.4 핵심 메시지
**"K-Food 수출, 이제 쉬워집니다 — 출시 전 사전 신청하고 특별 혜택을 받으세요"**

### 3.3 Section 2: Pain Points (문제 제기)

#### 목적

#### 콘텐츠 구성
**섹션 타이틀**: "해외 수출, 이런 고민하고 계시지 않나요?"

| 순번 | Pain Point | 상세 설명 |
|------|------------|----------|
| 1 | 어디서부터 시작해야 할지 모르겠어요 | 수출 인증, 통관, 라벨링... 복잡한 절차가 막막합니다 |
| 2 | 믿을 수 있는 바이어를 찾기 어려워요 | 검증되지 않은 바이어와의 거래는 불안합니다 |
| 3 | 소규모라 수출 비용이 부담돼요 | 대행사 비용, 인증 비용... 초기 투자가 걱정됩니다 |
| 4 | 해외 시장 정보가 부족해요 | 어떤 국가에서 우리 제품이 팔릴지 모르겠습니다 |

**전환 문구**: "XBridge가 이 모든 고민을 해결해 드립니다"

#### 비주얼 요소
- 각 Pain Point를 카드 형태로 표현

### 3.7 Section 6: FAQ (자주 묻는 질문)

#### 목적
사전 신청 관련 의문점 해소

#### 콘텐츠 구성
**섹션 타이틀**: "자주 묻는 질문"

| 질문 | 답변 |
|------|------|
| 사전 신청하면 비용이 발생하나요? | 아니요, 사전 신청은 완전 무료입니다. 정식 출시 후 서비스 이용 시에만 요금이 부과됩니다. |
| 사전 신청 후 반드시 서비스를 이용해야 하나요? | 아니요, 의무 사항이 아닙니다. 사전 신청은 관심 표현이며, 출시 시 이용 여부는 자유롭게 결정하실 수 있습니다. |
| 서비스는 언제 출시되나요? | 2026년 상반기 출시를 목표로 준비 중입니다. 정확한 일정이 확정되면 사전 신청자에게 가장 먼저 안내해 드립니다. |
| 사전 신청 혜택은 언제까지 유효한가요? | 정식 출시일로부터 3개월간 유효합니다. 출시 전 신청자 모두에게 동일하게 적용됩니다. |
| 베타 테스트에는 어떻게 참여하나요? | 사전 신청자 중 선착순으로 베타 테스터를 선정합니다. 선정되시면 별도로 안내 메일을 보내드립니다. |
| 제출한 정보는 어떻게 사용되나요? | 서비스 출시 안내, 혜택 제공, 서비스 개선을 위한 목적으로만 사용됩니다. 개인정보처리방침에서 자세한 내용을 확인하실 수 있습니다. |

#### 비주얼 요소
- 아코디언 형태 (클릭하여 펼침)
- 검색 기능 (선택적)
